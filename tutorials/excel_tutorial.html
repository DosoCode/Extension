Getting Started

  • To begin using Excel, open it from Microsoft Office or Microsoft 365.
  • If you don't have it, you can use the free online version at office.com.
  • Click "Blank Workbook" to start a new spreadsheet.
  • Each file in Excel is called a workbook, and each page inside it is called a worksheet.

The Excel Interface

When Excel opens, you will see rows (numbered 1, 2, 3...) and columns (labeled A, B, C...). Each square is called a cell. The toolbar at the top is called the Ribbon. It contains tabs like Home, Insert, Page Layout, and Formulas, each with different tools.

How to Enter Data

Click on any cell and start typing numbers or words. Press Enter or Tab to move to the next cell. You can also copy and paste data just like in Word.

Formatting Cells

Select the cells you want to format. Use the "Home" tab to change font, size, color, alignment, or add borders. You can also format numbers as currency, percentages, or dates.

Using Basic Formulas

Excel can calculate automatically. Type = followed by a formula. For example:

  • =A1+A2 adds values in cells A1 and A2.
  • =SUM(A1:A10) adds up numbers from A1 through A10.
  • =AVERAGE(B1:B5) finds the average of values in B1 through B5.

Creating Charts and Graphs

To make data easier to understand, highlight the cells you want to chart. Go to the "Insert" tab, then pick from Column, Line, or Pie charts. Excel will create a chart instantly.

Saving and Sharing Files

Click "File" > "Save As" to save your workbook. Choose a location on your computer or OneDrive. Use "Share" to send the file to others by email or link.

Helpful Tips

  • Use Ctrl + Z to undo a mistake.
  • Double-click between column letters to auto-fit the column width.
  • Freeze Panes under the "View" tab keeps headers visible while you scroll.
  • Use filters (under Data tab) to quickly find specific information.

Extra Stuff

This section includes some useful extras:

  • Conditional Formatting: Highlight cells automatically based on rules (e.g., red for low numbers, green for high numbers).
  • Sorting: You can sort data alphabetically, by numbers, or by date.
  • Pivot Tables: Summarize large amounts of data easily with pivot tables. They let you group, filter, and analyze without changing your raw data.
  • Keyboard Shortcuts: Save time by using Ctrl + C (copy), Ctrl + V (paste), Ctrl + S (save).
  • Templates: Excel has pre-made templates for budgets, calendars, and invoices under "File" > "New."
